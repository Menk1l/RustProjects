## ⚙️ Requirements

- Rust 1.XX or higher
- Cargo for dependency management
- A text editor like Visual Studio Code or IntelliJ Rust

## 📝 Usage

Clone this repository and build the project using Cargo:

```bash
git clone https://github.com/mwnkil/RustProjects.git
cd RustProjects
cargo build
```

Run the application:

```bash
cargo run
```
